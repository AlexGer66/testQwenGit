# Менеджер задач (PHP + MySQL)

Простое учебное приложение для управления задачами.

## Структура проекта
- `database.sql` — SQL-дамп для создания базы данных и таблицы
- `config.php` — настройки подключения к БД
- `db.php` — файл подключения к базе данных через PDO
- `index.php` — главная страница (список задач + форма добавления)
- `add_task.php` — обработчик добавления новой задачи
- `style.css` — базовые стили

## Установка
1. Создайте базу данных, импортировав `database.sql`
2. Отредактируйте `config.php` под ваши данные доступа к MySQL
3. Разместите файлы на веб-сервере с поддержкой PHP
4. Откройте `index.php` в браузере
